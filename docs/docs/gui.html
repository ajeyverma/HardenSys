Graphical User Interface (GUI)

Complete documentation for using the HardenSys GUI tool for interactive security management.

Note: The GUI provides an intuitive interface for users who prefer visual interaction over command-line operations.

Overview

The HardenSys GUI provides a user-friendly graphical interface for running security compliance checks, generating reports, and managing Windows security policies. It's designed for users who prefer visual interaction over command-line operations.

Key Features

  • Intuitive graphical interface with category selection
  • Real-time progress tracking and status updates
  • Interactive report generation and viewing
  • Backup and restore functionality
  • Settings configuration and customization
  • Keyboard shortcuts for power users

Launching the GUI

Command Line 
python HardenSys_gui.py

Interface Navigation

Main Window Layout

The GUI consists of several key components:

Menu Bar

Access to File, Edit, View, Tools, and Help menus

Toolbar

Quick access to common functions like Run Checks, Generate Report, and Settings

Category Panel

Select security categories and subcategories for compliance checks

Progress Panel

Real-time progress tracking and status updates during checks

Results Panel

Display compliance check results and detailed information

Status Bar

Current status, progress information, and system notifications

Menu Structure

File Menu

  • New Check: Start a new compliance check
  • Open Report: Open an existing compliance report
  • Save Report: Save current results to file
  • Export: Export results in various formats
  • Exit: Close the application

Edit Menu

  • Select All: Select all parameters
  • Clear Selection: Clear current selection
  • Find: Search for specific parameters
  • Preferences: Open settings dialog

View Menu

  • Refresh: Refresh the current view
  • Show Details: Toggle detailed information display
  • Full Screen: Toggle full screen mode
  • Zoom: Adjust display zoom level

Tools Menu

  • Run Checks: Execute selected compliance checks
  • Generate Report: Create compliance report
  • Backup Policies: Backup current security policies
  • Restore Policies: Restore from backup

Help Menu

  • User Guide: Open user documentation
  • Keyboard Shortcuts: View shortcut reference
  • About: Application information
  • Check Updates: Check for application updates

Running Compliance Checks

Selecting Categories

Choose which security categories to check:

1

Select Category

Choose a security category from the dropdown menu in the Category Panel

2

Choose Subcategory

Select specific subcategories within the chosen category

3

Configure Options

Set any additional options like output format or detailed logging

4

Run Checks

Click "Run Checks" button or use Ctrl+R keyboard shortcut

Check Types

Individual Checks

Run checks for specific parameters or subcategories

GUI Steps 
1. Select specific parameter
2. Click "Run Check" button
3. View results in Results Panel

Category Checks

Run checks for entire security categories

GUI Steps 
1. Select security category
2. Choose subcategories
3. Click "Run Category Check"
4. Monitor progress in Progress Panel

Full System Checks

Run comprehensive checks across all security parameters

GUI Steps 
1. Select "All Categories"
2. Click "Run Full Check"
3. Monitor progress and results
4. Generate comprehensive report

Progress Monitoring

The Progress Panel provides real-time feedback during compliance checks:

  • Progress Bar: Visual indication of completion percentage
  • Status Messages: Current operation and status updates
  • Check Counter: Number of checks completed vs. total
  • Time Elapsed: Duration of current check session
  • Success/Failure Count: Running tally of results

Report Generation

Generating Reports

Create comprehensive compliance reports from your check results:

1

Complete Checks

Run compliance checks and wait for completion

2

Review Results

Examine results in the Results Panel

3

Generate Report

Click "Generate Report" or use Ctrl+G shortcut

4

Choose Format

Select report format (HTML, PDF, or Text)

5

Save Report

Choose location and filename for the report

Report Formats

HTML Reports

Interactive web-based reports with detailed formatting

  • Clickable navigation
  • Expandable sections
  • Color-coded results
  • Print-friendly layout

PDF Reports

Professional PDF documents for formal documentation

  • Professional formatting
  • Table of contents
  • Executive summary
  • Detailed findings

Text Reports

Plain text reports for simple viewing and processing

  • Simple format
  • Easy to parse
  • Small file size
  • Universal compatibility

Report Contents

All report formats include:

  • Executive Summary: High-level overview of compliance status
  • Detailed Results: Complete results for each parameter checked
  • Recommendations: Suggested actions for non-compliant settings
  • Statistics: Success rates, failure counts, and timing information
  • Metadata: Report generation date, time, and system information

Settings and Configuration

Application Settings

Configure the GUI behavior and appearance:

General Settings

  • Language: Select interface language
  • Theme: Choose light or dark theme
  • Font Size: Adjust text size for readability
  • Auto-save: Automatically save reports
  • Check Updates: Automatically check for updates

Check Settings

  • Default Categories: Set default categories to check
  • Timeout: Set timeout for individual checks
  • Retry Attempts: Number of retry attempts for failed checks
  • Verbose Logging: Enable detailed logging
  • Parallel Checks: Run multiple checks simultaneously

Report Settings

  • Default Format: Set default report format
  • Output Directory: Default location for reports
  • Include Timestamps: Add timestamps to reports
  • Compress Reports: Compress large reports
  • Email Reports: Email reports automatically

Security Settings

  • Require Admin: Require administrator privileges
  • Backup Before Changes: Create backup before modifications
  • Confirm Changes: Confirm before applying changes
  • Audit Logging: Log all security changes
  • Encrypt Reports: Encrypt sensitive reports

Keyboard Shortcuts

Power users can use keyboard shortcuts for faster operation:

Shortcut Action Description
Ctrl+N New Check Start a new compliance check
Ctrl+R Run Checks Execute selected compliance checks
Ctrl+G Generate Report Create compliance report
Ctrl+S Save Report Save current results
Ctrl+O Open Report Open existing report
Ctrl+B Backup Backup security policies
Ctrl+Shift+R Restore Restore from backup
F1 Help Open help documentation
F5 Refresh Refresh current view
Esc Cancel Cancel current operation

Best Practices

Tip: Configure settings according to your organization's security requirements and workflow.
  • Enable automatic backups before making changes
  • Set appropriate timeouts for your environment
  • Configure email notifications for important events
  • Use keyboard shortcuts for improved efficiency
  • Regularly update the application for security patches
  • Test settings in a non-production environment first

Graphical User Interface (GUI)

Complete documentation for using the HardenSys GUI tool for Linux interactive security management.

Linux GUI: The graphical interface provides an intuitive way to manage Linux security configurations.

Overview

The HardenSys GUI is designed for Linux environments and provides a user-friendly interface for managing security configurations. It offers visual feedback, easy navigation, and comprehensive reporting capabilities.

Linux-Specific Features

  • Kernel parameter management interface
  • Systemd service configuration
  • File system permission management
  • Network security configuration
  • SSH configuration interface
  • PAM authentication management

Navigation

Main Interface

The Linux GUI interface includes:

  • Category Selection: Choose from Linux-specific security categories
  • Parameter List: View all parameters for the selected category
  • Status Indicators: Visual feedback on compliance status
  • Action Buttons: Run checks, apply fixes, generate reports

Linux Categories

Filesystem Configuration

Kernel modules and partition settings

Package Management

Bootloader and process hardening

Services Configuration

Server and client service management

Network Configuration

Network parameters and firewall settings

Access Control

SSH, PAM, and user account controls

Running Checks

Starting the GUI

Launch the HardenSys GUI on Linux:

Linux Terminal 
python3 HardenSys_gui.py

Running Security Checks

1

Select Category

Choose a Linux security category from the dropdown menu

2

Review Parameters

Examine the list of security parameters for the selected category

3

Run Checks

Click "Run Checks" to start the compliance verification

4

Review Results

Examine the compliance status and recommended actions

Linux-Specific Checks

  • Kernel Module Verification: Check for disabled kernel modules
  • File System Permissions: Verify partition and file permissions
  • Service Status: Check systemd service configurations
  • Network Security: Verify firewall and network settings
  • SSH Configuration: Check SSH security settings
  • PAM Settings: Verify authentication configurations

Reports

Generating Reports

Generate comprehensive compliance reports for Linux systems:

HTML Reports

Linux Terminal 
# Generate HTML report
python3 HardenSys_gui.py --report html --output linux_compliance.html

PDF Reports

Linux Terminal 
# Generate PDF report
python3 HardenSys_gui.py --report pdf --output linux_compliance.pdf

Report Contents

  • Executive Summary: Overall compliance status
  • Category Breakdown: Detailed results by security category
  • Parameter Details: Individual parameter compliance status
  • Recommendations: Suggested remediation actions
  • System Information: Linux distribution and version details

Settings

Configuration Options

Customize the GUI behavior for Linux environments:

Display Settings

  • Theme: Choose between light and dark themes
  • Font Size: Adjust text size for better readability
  • Language: Select interface language

Security Settings

  • Auto-backup: Automatically backup configurations before changes
  • Confirmation Prompts: Require confirmation for critical operations
  • Logging Level: Set verbosity of operation logs

Linux-Specific Settings

  • Default Editor: Choose text editor for configuration files
  • Terminal Emulator: Select preferred terminal application
  • Package Manager: Specify package manager (apt, yum, dnf)

Keyboard Shortcuts

Ctrl + R Run security checks
Ctrl + G Generate report
Ctrl + S Save configuration
Ctrl + Q Quit application
