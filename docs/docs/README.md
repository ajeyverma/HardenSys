# HardenSys Documentation

```

├── index.html              # Main documentation homepage
├── cli.html                # CLI interface documentation
├── gui.html                # GUI interface documentation

├── js/

│   └── parameters.js       # Parameters page JavaScript
└── README.md               # This file
```
