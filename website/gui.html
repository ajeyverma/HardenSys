Graphical User Interface (GUI) Guide

Complete documentation for using the HardenSys GUI tool

Overview

The HardenSys GUI tool provides an intuitive graphical interface for managing Windows security policies, running compliance checks, and generating reports. It's perfect for users who prefer a visual interface over command-line operations.

Note: The GUI tool requires Administrator privileges for full functionality. Some features may be limited without proper permissions.

Getting Started

Launching the GUI

# Run the GUI application python gui.py # Or use the batch file (Windows) run_gui.bat

First Launch

When you first launch the GUI, you'll see the main interface with the following sections:

  • Navigation Panel: Browse through different security categories
  • Parameter List: View and select specific security parameters
  • Action Buttons: Run checks, generate reports, and manage settings
  • Status Bar: Display current operation status and results

Running Compliance Checks

Individual Parameter Checks

To check a specific parameter:

  1. Navigate to the desired category in the left panel
  2. Select the specific parameter you want to check
  3. Click the "Check" button next to the parameter
  4. View the results in the status area

Category-wide Checks

To check all parameters in a category:

  1. Select the category (e.g., "Account Policies")
  2. Click the "Check All" button for that category
  3. Monitor the progress in the status bar
  4. Review the results when complete

Full System Check

To run a complete compliance check:

  1. Click the "Run All Checks" button in the toolbar
  2. Wait for all 121 parameters to be checked
  3. Review the comprehensive results
  4. Generate a report if needed

Report Generation

HTML Reports

Generate detailed HTML reports with visual formatting:

  1. Run your desired compliance checks
  2. Click "Generate Report" in the toolbar
  3. Select "HTML Format"
  4. Choose the output location
  5. Click "Generate" to create the report

PDF Reports

Create professional PDF reports for documentation:

  1. Complete your compliance checks
  2. Click "Generate Report" in the toolbar
  3. Select "PDF Format"
  4. Choose the output location
  5. Click "Generate" to create the PDF

Report Contents

Reports include:

  • Executive summary with overall compliance percentage
  • Detailed results for each parameter checked
  • Before and after values for changed settings
  • Recommendations for non-compliant parameters
  • Timestamp and system information

Backup and Restore

Creating Backups

Before making changes, always create a backup:

  1. Click "Backup" in the toolbar
  2. Select the backup location
  3. Choose a descriptive filename
  4. Click "Create Backup"

Restoring from Backup

To restore from a previous backup:

  1. Click "Restore" in the toolbar
  2. Browse to the backup file (.inf format)
  3. Review the backup details
  4. Click "Restore" to apply the settings
Warning: Restoring from backup will overwrite current security settings. Ensure you have a recent backup before proceeding.

Settings and Configuration

Application Settings

Access settings through the "Settings" menu:

  • Default Report Format: Choose HTML or PDF as default
  • Auto-backup: Automatically create backups before changes
  • Logging Level: Set verbosity of operation logs
  • Theme: Choose between light and dark themes

Parameter Configuration

Customize parameter behavior:

  • Severity Levels: Set high/medium/low severity for parameters
  • Custom Values: Define custom values for specific parameters
  • Exclusions: Exclude certain parameters from checks
  • Grouping: Organize parameters into custom groups

Keyboard Shortcuts

Troubleshooting

Common Issues

Issue: "Access Denied" errors
Solution: Run the application as Administrator by right-clicking and selecting "Run as administrator"
Issue: GUI not responding
Solution: Check if the application is running with proper permissions and restart if necessary
Issue: Report generation fails
Solution: Ensure you have write permissions to the output directory and sufficient disk space

Performance Tips

  • Close unnecessary applications before running full system checks
  • Use category-specific checks for faster results
  • Enable auto-backup only when necessary to improve performance
  • Clear old reports periodically to free up disk space

Best Practices

Before Making Changes

  1. Always create a backup of current settings
  2. Run a full compliance check to understand current state
  3. Review the results and plan your changes
  4. Test changes on a non-production system first

Regular Maintenance

  1. Run weekly compliance checks
  2. Generate monthly compliance reports
  3. Review and update security policies as needed
  4. Keep backups of all configuration changes

Security Considerations

  • Store backups in secure locations
  • Use strong passwords for any authentication
  • Regularly update the tool and dependencies
  • Monitor compliance reports for security trends
