Installation Instructions

Quick Installation

# 1. Download the complete package # 2. Extract to desired location (e.g., C:\SecurityCompliance) # 3. Open Command Prompt as Administrator # 4. Navigate to the extracted folder cd C:\SecurityCompliance # 5. Install Python dependencies pip install -r requirements.txt # 6. Run the tool python gui.py

Manual Installation

  1. Download Python: Install Python 3.7+ from python.org
  2. Download the tool: Get the complete package or individual components
  3. Extract files: Extract to a folder of your choice
  4. Install dependencies: Run pip install -r requirements.txt
  5. Verify installation: Run python compliance_cli.py --help

Dependencies

# Core dependencies (included in requirements.txt) PySide6>=6.0.0 # GUI framework reportlab>=3.0.0 # PDF report generation PyPDF2>=3.0.0 # PDF manipulation Pillow>=8.0.0 # Image processing python-dotenv>=0.19.0 # Environment variables

Quick Start

GUI Quick Start

# Run the GUI application python gui.py # Or use the batch file (Windows) run_gui.bat

CLI Quick Start

# Run all compliance checks python compliance_cli.py # Run specific category python compliance_cli.py --heading "Account Policies" # Generate report python compliance_cli.py --output report.html

PowerShell Quick Start

# Run with PowerShell launcher .\run_compliance.ps1 # With parameters .\run_compliance.ps1 -Heading "Account Policies" -Output "report.html"
